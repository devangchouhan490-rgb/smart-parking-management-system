# smart-parking-management-system
Smart Parking Management System in C language

This is a C language based Smart Parking Management System.

## Features
- Vehicle number validation
- Automatic slot assignment
- Manual slot booking option
- Parking fee calculation
- Overstay (24+ hours) charge update
- View and delete parking records

## Technologies Used
- C Language
- File handling
- Time library
